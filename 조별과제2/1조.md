# 1. 공기압 회로
<img width="846" height="366" alt="가" src="https://github.com/user-attachments/assets/8541369a-986e-45c6-ba5c-18c4ee8745bb" />

실린더 A, B는 각각 5/2 솔레노이드 밸브(Y1, Y2)로 구동함.  
리미트 스위치 LS1~LS4는 실린더의 위치를 감지하여 전기회로에 피드백을 줌.  
| 감지기 |	역할                      |
|--------|---------------------------|
| LS1	| A 실린더 전진 말단 감지 (A+) |
| LS2	| A 실린더 후진 말단 감지 (A-) |
| LS3	| B 실린더 전진 말단 감지 (B+) |
| LS4	| B 실린더 후진 말단 감지 (B-) |

# 2. 전기회로도
<img width="905" height="426" alt="나" src="https://github.com/user-attachments/assets/5eacd82a-fb76-4b84-b2ce-8a9c89f555d8" />

PB1: 연속 운전 버튼 → 한 번 누르면 1 사이클 자동 수행  
PB2: 카운터 초기화 버튼  
K1~K5: 중간 릴레이로 각 단계에서 신호를 유지 및 전달함  
Y1, Y2: 솔레노이드 밸브, 실린더 A와 B를 제어  
C1: 카운터, 3사이클 후 동작 정지 (리셋 가능)  

# 3. 변위단계선
<img width="582" height="305" alt="다" src="https://github.com/user-attachments/assets/ee808dcc-00ab-48f9-b442-f29132d33ec9" />

변위선도를 통해 실린더의 순차적인 동작을 확인할 수 있음.  
### 동작 순서 (1 사이클)  
**1.** A 전진 (A+) → LS1 감지  
**2.** A 후진 (A−) → LS2 감지  
**3.** B 전진 (B+) → LS3 감지  
**4.** B 후진 (B−) → LS4 감지  
**5.** 카운터 1 증가 (C1 = 1)
   
**→** 이 과정을 3회 반복하면 카운터 조건 만족 **→** 자동 정지.

# 4. 동작설명

| 스위치 | 기능 | 설명 |
|-------|------|-------|
| PB1 (연속 운전)	| 자동 사이클 시작	| 1회 ON → 3사이클 수행 후 정지 |
| PB2 (카운터 초기화) | C1 초기화	    | 사이클 수를 0으로 리셋 |

유량제어밸브로 실린더 속도를 약 50%로 조절.  
1회 ON 시 자동으로 A+ → A− → B+ → B− 순서로 3회 반복 동작 후 정지.  
카운터 C1이 3에 도달하면 릴레이 차단되어 동작 중지됨.


# 5. 동작순서
<img width="905" height="426" alt="나" src="https://github.com/user-attachments/assets/a48488af-22ef-4dee-80d3-70d131404c15" />

**1.** PB1 누름 → K1 동작 (시작)  
**2.** LS3 감지 → K2 → Y1 (A+)  
**3.** LS1 감지 → K3 → Y1 OFF, Y2 ON (A-)  
**4.** LS2 감지 → K4 → Y2 ON (B+)  
**5.** LS4 감지 → K5 → Y2 OFF (B-)  
**6.** LS3 감지 시 C1 카운터 증가  
**7.** C1 = 3이면 회로 차단 → 정지

| 순번 | 조건(입력 상태) | 출력 상태 | 동작 설명 |
|------|----------------|---------|-------------|
| 1 | PB1 ON |	K1 Set | 연속 운전 시작 |
| 2 |K1 ON & LS3 ON | K2 Set | A 실린더 전진 조건 완료 |
| 3 |LS1 ON |	K3 Set |	A 실린더 전진 완료 → A 후진 시작 |
| 4 |LS2 ON |	K4 Set |	A 실린더 후진 완료 → B 전진 시작 |
| 5 |LS4 ON |	K5 Set |	B 실린더 전진 완료 → B 후진 시작 |
| 6 |LS3 ON |	C1 카운터 +1 | 사이클 1회 완료 |
| 7 |C1 < 3 |	순번 2로 반복 |	3사이클 미만이면 반복 |
| 8 |C1 = 3 |	K1 Reset |	자동 정지 |
| 9 |PB2 ON |	C1 Reset |	카운터 초기화 |
